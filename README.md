# IPTV Desktop Player

A complete desktop IPTV player built with Electron.js. Supports Xtream Codes API, M3U/M3U8 playlists, HLS.js streaming, Arabic/English UI, EPG, and favorites.

---

## Features

- **Xtream Codes API** — Live TV, Movies, Series via full API integration
- **M3U/M3U8** — Load playlists from URL or local file
- **HLS.js** — Adaptive streaming for m3u8 streams; native fallback for mp4/ts/mkv
- **EPG** — XMLTV guide with now/next info
- **Favorites** — Persist channels, movies, series across sessions
- **Multiple Accounts** — Switch between any number of saved accounts
- **Bilingual UI** — Full Arabic (RTL) and English support
- **Logo Caching** — Downloads and caches channel logos locally
- **Search & Filter** — Per-view search + category filter tabs
- **Dark/Light Theme** — Switchable in Settings
- **Playlist Refresh** — Re-fetch playlist without re-entering credentials

## Quick Start (Local Development)

### Prerequisites

- **Node.js** 18+ ([nodejs.org](https://nodejs.org))
- **npm** or **yarn** (comes with Node.js)

### 1. Install dependencies

```bash
cd iptv-player
npm install
```

This installs:
- `electron` — desktop app runtime
- `electron-builder` — packaging / installer builder
- `hls.js` — HLS stream playback

### 2. Run in development mode

```bash
npm start
```

Or with DevTools enabled:

```bash
npm run dev
```

The app window opens immediately. You can inspect the renderer with DevTools from the `View` menu (dev mode only).

## Building the Windows Installer (.exe)

### Prerequisites

- Windows 10/11 **or** a machine with Wine (for cross-compilation)
- Node.js 18+

### 1. Install dependencies

```bash
npm install
```

### 2. Build installer (NSIS + Portable)

```bash
npm run build
```

This produces **two files** inside `dist/`:

| File | Description |
|------|-------------|
| `IPTV Desktop Player Setup 1.0.0.exe` | Full installer (NSIS) — installs to Program Files with Start Menu / Desktop shortcuts |
| `IPTV-Desktop-Player-Portable.exe`    | Standalone portable — no installation required |

### 3. Build installer only

```bash
npm run build:dir    # Unpacked directory (fast, for testing)
npm run build        # Full NSIS installer + portable
```

## App Icon

Place your icon files in `build/`:
- `build/icon.ico` — Windows icon (256×256 minimum, ICO format)
- `build/icon.png` — PNG fallback (512×512 recommended)

You can convert a PNG to ICO using tools like [icoconvert.com](https://icoconvert.com) or ImageMagick:

```bash
magick convert icon.png -resize 256x256 icon.ico
```

## Data Storage Location

All user data is stored in Electron's `userData` directory:

| Platform | Path |
|----------|------|
| Windows  | `%APPDATA%\IPTV Desktop Player\data\` |
| macOS    | `~/Library/Application Support/IPTV Desktop Player/data/` |
| Linux    | `~/.config/IPTV Desktop Player/data/` |

Files:
- `accounts.json` — saved accounts (credentials stored locally)
- `favorites.json` — favorited channels, movies, series
- `settings.json` — app settings
- `logos/` — cached channel logos

## Distribution

After building, share either:

1. **The NSIS installer** (`IPTV Desktop Player Setup 1.0.0.exe`) — end-users run this to install

## Troubleshooting

| Issue | Fix |
|-------|-----|
| Stream won't play | Check that the stream URL is reachable. Try pasting it in VLC to confirm. |
| CORS / network errors | The app uses Electron's Node.js `http/https` module — CORS is bypassed by design. |
| Logo not loading | Check the logo URL. The app caches logos on first load. |
| App won't start | Run `npm install` again. Check Node.js version ≥ 18. |
| Can't build .exe on Linux/Mac | Install `wine` or build on a Windows machine. |
